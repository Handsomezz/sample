## INSTALL Sample

Install [MetacelloPreview 1.0-beta.32][1], then:

```Smalltalk

//AL PACINO


Metacello new
  configuration: 'Sample';
  version: #stable;
  repository: 'github://dalehenrich/sample:configuration';
  load.
```
